# Kumar Saurav Portfolio

Gen Z-style responsive portfolio built with React.js, CSS3, Vite, Three.js, and real portfolio assets.

## What's Included

- Real resume details: email, phone, Dehradun, BCA, skills, experience, certifications, strengths.
- GitHub link and public project cards from `github.com/xmx7hsgr2z-alt`.
- Profile photo from the provided image.
- Resume PDF in `public/Kumar_Saurav_Resume.pdf`.
- Freelance section using six downloaded Dropbox transfer projects:
  - Impulse Classes
  - Scholars Hub Defence Institute
  - Indira Homeopathy Clinic
  - Ivory 32 Dental Clinic
  - Sri Ram Medical Centre
  - Vision Institute
- Scroll reveal transitions, hover transitions, 3D hero background, bold neon UI.

## Run Locally

```bash
npm.cmd install
npm.cmd run dev
```

Current dev server:

```text
http://localhost:5173/
http://192.168.0.108:5173/
```

## Production Build

```bash
npm.cmd run build
npm.cmd run preview -- --host 0.0.0.0
```

Production files are generated in `dist/`.
